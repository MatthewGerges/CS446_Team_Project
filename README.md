# Stellar

An app to split cost between roommates with an integrated to-do list for the roommates to complete.

The backend for this project is maintained in a separate repository for readability and maintainability. This also provides better security for secrets and API keys. A link to the backend repo is included below, and invitations have been sent to the teaching team. Please contact Daniel Molina with any issues accessing it.
